# Elite Hire Tools Website

A simple **React-based website for hiring tools and equipment**, allowing customers to browse available tools and quickly contact the business through **WhatsApp with a pre-filled enquiry message**.

## Features

* 🔧 **Tool Catalogue** – Displays available tools with images, descriptions, and prices
* 🔍 **Search Tools** – Users can search tools by name
* 🗂 **Filter by Category** – Filter tools by category (e.g., Construction, Electrical, Power)
* 💬 **WhatsApp Enquiries** – Clicking a tool opens WhatsApp
* 📱 **Responsive Design** – Works on both desktop and mobile devices

## How It Works

1. A user opens the website.
2. They browse the available tools.
3. They can **search** or **filter** tools to find what they need.
4. When a user clicks a tool, they are redirected to our **WhatsApp** 
```
5. The message is sent directly to the business WhatsApp number.

## Technologies Used

* **React.js**
* **HTML5**
* **CSS3**
* **JavaScript**
* **WhatsApp Click-to-Chat API**
